# Banking Transactions Analytics Dashboard

A complete Power BI dashboard analyzing banking transactions, spending patterns, and transaction performance using interactive visuals and clear insights.

---

## Project Overview
This project analyzes 60 banking transactions across multiple categories and months.  
The goal is to understand:
- Spending behavior  
- Category-wise distribution  
- Monthly trends  
- Transaction success vs failure  
- Key KPIs like total amount, average transaction, and failure rate  

---

## Key Features

### Dashboard Page
- Total Amount, Total Transactions, Average Transaction, Failed %  
- Category slicer  
- Month slicer  
- Monthly trend bar chart  
- Category-wise donut chart  
- Transaction status breakdown  
- Data quality indicator & last refresh timestamp  

### Insights Page
- Top 3 & Bottom 3 categories  
- Monthly trend insights  
- Spending behavior summary  
- Transaction performance analysis  
- Clean text-based insights

## Key Insights

### Spending Insights
- Total spending: **30.23K**  
- Highest spending category: **Rent (29.78%)**  
- Other major categories: Groceries, Shopping, Salary  

### Transaction Insights
- **93.33%** transactions completed  
- Only **0.03%** failure rate  
- Average transaction value: **503.76**  

### Monthly Trend
- Spending peaks in **March–May**  
- Lowest activity in **January**
